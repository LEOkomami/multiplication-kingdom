# ממלכת הכפל - הרפתקת חיות קסומה 🦄✨

ממלכת הכפל היא אפליקציית משחק דינמית ומרהיבה לילדים, המעוצבת לתרגול לוח הכפל בצורה מהנה, מעצימה ולא מאיימת.

## 🚀 מה חדש בממלכה?

### 1. חווית אפליקציה (PWA)
המשחק מוגדר כ-**Progressive Web App**. ניתן להוסיף אותו ל"מסך הבית" בטאבלט או בטלפון (אייפד או אנדרואיד) והוא יפתח כאפליקציה מלאה ללא שורת כתובת, עם תמיכה בעבודה לא מקוונת (Offline).

### 2. טבלת האלופים (Leaderboard) 🏆
תחרות חיובית בממלכה! נוסף עמוד "טבלת האלופים" המציג את הדירוג של כל הפרופילים המקומיים לפי ניקוד. ניתן לראות כמה חיות כל אחת אספה וכמה תאים נכבשו.

### 3. אתגרי קסם חזותיים 🌟
במקום רק להזין מספרים, כל תא בלוח הוא "משימה" חזותית:
- **גירוד הקסם**: מגרדים את העננים כדי לחשוף את החיה.
- **פיצוץ בועות**: מפוצצים בועות נוצצות.
- **כוכבי המזל**: תופסים כוכבים קופצים לפני שהם נעלמים.
- **מצאו את התואם**: אתגר זיכרון ומהירות.

### 4. שיטת הניקוד והבונוס
- **כיבוש רגיל**: מעבר של מיני-משחק מזכה בנקודה אחת וחיה לאוסף.
- **בונוס ידע**: אם פותרים את תרגיל הכפל מספרית (בתיבת הבונוס) לפני סיום המשחק, זוכים ב-**3 נקודות!**

## 💻 חומרה ודפדפן
- **אופטימיזציה**: המשחק מותאם במיוחד ל**טאבלטים** ולמחשבים אישיים.
- **חסימת ניידים**: בניידים קטנים תופיע הודעה שהמשחק מיועד למסכים גדולים יותר לחוויית למידה מיטבית.
- **מיתוג**: A KONAMI APP.

## 🛡️ פרטיות וטכנולוגיה
- **ללא שרת**: כל המידע (פרופילים, נקודות, חיות) נשמר ב-**LocalStorage** של הדפדפן שלך בלבד.
- **פרטיות מלאה**: אין איסוף מידע, אין הרשמה ואין העלאת נתונים לענן.
- **טכנולוגיה**: Vanilla JS, HTML5 Canvas, CSS3 Grid, Service Workers.
